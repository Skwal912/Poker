Configuration Spotify

📋 Setup en 4 étapes :
  1. Va sur developer.spotify.com/dashboard et crée une app (n'importe quel nom)
  2. Dans les settings de l'app, ajoute comme Redirect URI :
    ⚠️ Si tu vois file:// ci-dessus : l'OAuth Spotify refuse file://. Lance le fichier via un mini-serveur :
    python3 -m http.server 8080
    puis ouvre http://127.0.0.1:8080/poker-manager.html et tu verras le bon redirect URI. Une fois connecté, tu pourras revenir en file:// et le refresh token continuera à fonctionner.
  3. Copie le Client ID de ton app et colle-le ci-dessous
  4. Connecte-toi · Lance ensuite Spotify sur ton téléphone, ordi ou enceinte — le poker manager le pilote à distance
💡 Premium requis pour les contrôles (play/pause/skip/volume). C'est une limitation de l'API Spotify, pas de cette app.
